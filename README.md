# orgpulse.io

Marketing and documentation site for [OrgPulse](https://marketplace.atlassian.com) — a Jira Cloud app that diagnoses engineering team health.

Built with [Astro](https://astro.build). Deployed to GitHub Pages at [orgpulse.io](https://orgpulse.io).

## Commands

| Command           | Action                                      |
| :---------------- | :------------------------------------------ |
| `npm install`     | Install dependencies                        |
| `npm run dev`     | Start local dev server at `localhost:4321`  |
| `npm run build`   | Build to `./dist/`                          |
| `npm run preview` | Preview build locally                       |
